# my-monorepo
My monorepo used to start everything :-)

Huge thanks to [midday](https://github.com/midday-ai/midday) !!!

## Tech stack
- [bun](https://bun.sh/) - package manager and runtime
- [biome](https://turborepo.dev/docs/reference/configuration) - for linting and formatting
- [turbo](https://turborepo.dev/docs/reference/configuration) - monorepo management
- [typescript](https://www.typescriptlang.org/docs/handbook/tsconfig-json.html) - main programming language

- [shadcn](https://ui.shadcn.com/docs/monorepo) - component library
- [tailwindcss](https://tailwindcss.com/docs/installation) - css framework
- [tanstack](https://tanstack.com/start/latest/docs/framework/react/quick-start) - react query and router
